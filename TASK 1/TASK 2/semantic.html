Core concept

HyperText: Refers to links (hyperlinks) that connect web pages to one another,
either within a single site or across different websites.
Markup Language: Uses a series of "tags" to annotate text and images so a web browser knows how to display them.
Elements: The building blocks of HTML, consisting of an opening tag, content, and a closing tag

HTML(HyperText Markup Language) is the standard language used to create and structure web pages.

It acts as the "skeleton" of a website, using a system of tags and elements to define content like
headings, paragraphs, links, and images
